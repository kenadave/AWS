1) AWS Cloud Map is a cloud resource discovery service. With Cloud Map, you can define custom names for your application resources, and it maintains the updated location of these dynamically changing resources

2) AWS Private 5G is for building your own cellular network inside your facility, while AWS Wavelength is for putting your apps inside a major carrier's (like Verizon) 5G network.

3) Private 5G uses the AWS Private 5G Console to order hardware and manage your private spectrum (like CBRS in the US).
Wavelength works by extending your VPC into a Wavelength Zone and using a Carrier Gateway to talk to the mobile network.
